`libtiff.so.3`, `libjpeg.so.62`, `libpng12.so.0`, the krb5 sonames
(`libgssapi_krb5.so.2`, `libkrb5.so.3`, `libk5crypto.so.3`, `libcom_err.so.2`),
`libz.so.1`, `libstdc++.so.6`, `libfontconfig.so.1`.

`libtiff libjpeg-turbo libpng cups-libs krb5-libs fontconfig libstdc++`
(plus `binutils gdb`). Note CentOS 6's `libtiff` is 3.9.x → `libtiff.so.3`,
which matches what the 2007 build links against. No extra packages are needed.
